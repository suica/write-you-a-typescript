## 类型体操的本质

注意：TypeScript类型编程从本质上来说，就是使用类型系统这种非常受限的编程语言来解答问题。它和我们在运行时编程，没有本质的区别；此外，在日常的编程工作中，需要适当使用类型体操的技巧，在追求效率的工程实践和追求优雅的审美过程之间达成平衡。
