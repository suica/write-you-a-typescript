TAT-Sub的设计和实现

TAT-Sub​

TAT-Sub是基于TAT-STLC的类型检查器，具有前者的全部功能，因此保留所有TAT-STLC的类型规则。

此外，还增加了子类型的定型规则。

子类型定型规则​

自反性。

S<:S(S-Refl){S<:S} \tag{S-Refl}S<:S(S-Refl)

传递性。

S<:UU<:TS<:T(S-Trans){ S <: U \quad U <: T \over S <: T } \tag{S-Trans}S<:TS<:UU<:T​(S-Trans)

我们还引入了顶类型。

S<:Top(S-Top){S<:\text{Top}} \tag{S-Top}S<:Top(S-Top)

关于函数类型，我们使用之前推导出来的定型规则。

T1<:S1S2<:T2S1⇒S2<:T1⇒T2(S-Arrow){ T1 <: S1 \quad S2 <: T2 \over S1 \Rightarrow S2 <: T1 \Rightarrow T2 } \tag{S-Arrow}S1⇒S2<:T1⇒T2T1<:S1S2<:T2​(S-Arrow)

子类型定型规则（续）​

这是我们使用的对象类型的子类型定型规则，它引入了结构子类型。

{lii∈1..n}⊆{kjj∈1..m}kj=liimpliesSj<:Ti{kj:Sjj∈1..m}<:{li:Tii∈1..n}(S-Obj)\{l_i\quad^{i\in 1..n}\}⊆\{k_j\quad^{j\in 1..m}\} \\ \\ { k_j = l_i \quad implies \quad S_j <: T_i \over \{k_j:S_j\quad ^{j\in 1..m}\} <: \{l_i:T_i\quad ^{i\in 1..n}\} } \tag{S-Obj}{li​i∈1..n}⊆{kj​j∈1..m}{kj​:Sj​j∈1..m}<:{li​:Ti​i∈1..n}kj​=li​impliesSj​<:Ti​​(S-Obj)

这个规则下，我们有：

{a:Num, b:{c:Str}} <: {a:Num}
{a:Num, b:{c:Str}} <: {a:Num, b:{c:Top}}

我们还需要往我们的类型系统中加入安全替换原则。
